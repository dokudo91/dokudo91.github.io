確率分布の変換

ある確率分布の確率密度関数をpu(u)p_u(u)pu​(u)とする。この確率変数uをfで変換した確率変数をv=f(u)v=f(u)v=f(u)とする。 pup_upu​が連続分布で、v=f(u)v=f(u)v=f(u)が１対１に対応している時、確率変数vの確率密度関数pv(v)p_v(v)pv​(v)は

pv(v)=∣J∣pu(f−1(v)) p_v(v)=|J|p_u(f^{−1}(v)) pv​(v)=∣J∣pu​(f−1(v))

となる。ここでJはヤコビアンである。

Julia code

Juliaで計算結果を確認する。pup_upu​を２変量正規分布とする。それぞれ平均0、標準偏差1とする。 uは(u1,u2)(u_1,u_2)(u1​,u2​)のベクトル、vは(v1,v2)(v_1,v_2)(v1​,v2​)のベクトルとする。

using Distributions, StatsPlots
pu=MvNormal([0, 0],[1 0; 0 1])
contour(-0.5:0.01:0.5, -0.5:0.01:0.5, (x, y) -> pdf(pu, [x, y]), size=(500, 400), yrotation=90)
savefig(joinpath(@OUTPUT, "pu.svg"))
